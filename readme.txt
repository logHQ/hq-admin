== Description ==

HQ Admin is WordPress plugin for nice admin.

== Installation ==

1. Unzip 'hq-admin.zip'
2. Upload the 'hq-admin' folder to the '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Will it work on my site? =
It will if you have atleast WordPress 4.0
